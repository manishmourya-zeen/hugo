## 🎨 Design System

### Color Palette
- **Background:** `#FFFFFF` (White)
- **Text:** `#000000` (Black)  
- **Accent:** `#000000` (Black)
- **Borders:** `#E0E0E0` (Light Gray)

### Typography
- **Primary Font:** JetBrains Mono, Fira Code, Roboto Mono (monospace)

- `--spacing-xs: 0.5rem` (8px)
- `--spacing-sm: 1rem` (16px)
- `--spacing-md: 1.5rem` (24px)
- `--spacing-lg: 2rem` (32px)
- `--spacing-xl: 3rem` (48px)

### Responsive Breakpoints
- Desktop: 900px max-width container
- Tablet: Full width with padding
- Mobile: `max-width: 768px` - adjusted spacing
- Small Mobile: `max-width: 480px` - compact layout

## 🚀 Hugo Configuration (hugo.toml)

### Key Sections

**Outputs**
```toml
[outputs]
home = ["HTML"]
section = ["HTML", "RSS"]
page = ["HTML"]
```

**Navigation Menu**
```toml
[[menu.main]]
name = "Posts"
url = "/posts/"
weight = 2
```

**Parameters**
```toml
[params]
author = "Your Name"              # Site author
description = "..."               # SEO description
keywords = "..."                  # SEO keywords

[params.social]
github = "https://github.com/..."
linkedin = "https://linkedin.com/..."
twitter = "https://twitter.com/..."
email = "email@example.com"

[params.analytics]
googleAnalyticsID = "UA-XXXXXXXXX-X"

[params.resume]
downloadURL = "/resume.pdf"
```

## 🔧 How to Use

### Local Development

```bash
# Navigate to project
cd /home/ubuntu/devopsinfo.in

# Start development server (port 1313)
hugo server -D

# In browser: http://localhost:1313
```

### Create New Blog Post

```bash
hugo new posts/my-new-post.md
```

This creates:
```
content/posts/my-new-post.md
---
title: "My New Post"
date: 2026-04-30T10:00:00+00:00
draft: true
---
```

Edit the file and change `draft: false` to publish.

### Build for Production

```bash
hugo

# Output in /public directory
# Deploy the /public folder to your hosting
```

### Update Site Config

Edit `hugo.toml` to customize:
- Domain name (`baseURL`)
- Author information (`[params]`)
- Social media links (`[params.social]`)
- Google Analytics ID (`[params.analytics.googleAnalyticsID]`)
- Resume PDF location (`[params.resume.downloadURL]`)

## 📊 Front Matter (Page Metadata)

### Full Example
```yaml
---
title: "Post Title"
date: 2026-04-28
lastmod: 2026-04-29
draft: false
tags: ["tag1", "tag2", "tag3"]
categories: ["Technology"]
author: "Author Name"
description: "Short description for SEO"
weight: 1
---
```

### Commonly Used Fields
- `title` - Page/post title (required)
- `date` - Publication date (required)
- `draft` - Set to `false` to publish
- `tags` - Array of tags for categorization
- `author` - Post-specific author override
- `description` - Custom meta description
- `weight` - Sort order (lower = higher priority)

## 📈 Performance Optimizations

- **Lightweight CSS:** Vanilla CSS, ~3KB minified
- **No JavaScript:** Fast rendering, SEO-friendly
- **Google Fonts:** Async loading with `fetchpriority="low"`
- **No third-party themes:** Faster builds and better control

- **Responsive Design:** Mobile-first approach
- **Code Highlighting:** Built-in Hugo highlighting (no JS needed)
